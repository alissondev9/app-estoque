# App de Estoque em Ruby (Console)

Um aplicativo simples de controle de estoque desenvolvido em **Ruby** para execução via console. Permite gerenciar produtos de forma prática, sendo ideal para pequenos negócios ou como projeto de estudo em programação.

---

## Funcionalidades

- Cadastrar novos produtos
- Listar todos os produtos cadastrados
- Atualizar informações de produtos
- Remover produtos do estoque
- Interface simples e intuitiva via console

---

## Tecnologias

- Ruby
- Arquivos de texto ou arrays para armazenamento temporário (dependendo da implementação)
